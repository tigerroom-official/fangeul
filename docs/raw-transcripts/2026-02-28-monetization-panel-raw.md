## Topic 1: 버블 게이팅 타당성

### A조 선공

**민수익 (수익화 전문가):**

> "가치 인식이 먼저입니다."

버블 Pro 게이팅은 가치 인식 앵커링입니다. 배너 대비 ARPU 3~5배 향상 가능합니다. 킬러 피처를 무료 개방하면 광고 시청 동기가 없습니다.
- 근거 1: 유틸리티 앱 벤치마크 — Tasker, Bouncer 등 킬러 피처 게이팅 앱의 전환율 5~8% vs 배너 의존 앱 eCPM $0.3~0.5
- 근거 2: 무료 유저가 버블 가치를 체험하면 "이미 충분하다" → 전환 동기 소멸
- 근거 3: 경쟁자 없는 현 시점이 프리미엄 포지셔닝 최적 타이밍

**잭그로 (인디 앱 그로스):**

> "데이터가 말하게 하세요."

버블은 페인킬러(없으면 고통)입니다. 보상형 광고 = 결제 의향 측정 필터입니다. 경쟁 없을 때가 프리미엄 포지셔닝 최적 타이밍입니다.
- 근거 1: 페인킬러 앱은 게이팅해도 이탈률 낮음 (Switch앱 사례: 게이팅 후 DAU -12%, Revenue +180%)
- 근거 2: 보상형 광고 시청 빈도 = 결제 의향 프록시 → 데이터 기반 가격 최적화 가능
- 근거 3: 선점 효과 — 후발 주자가 무료로 들어오기 전에 유료 고객 기반 확보

### B조 반론

**하늘 (팬 UX 리서처):**

> "팬의 마음을 읽어야 합니다."

Weverse 라이브 방송 중 순간 마찰이 발생하면 "도움→방해" 인식 역전이 일어납니다. 써보지 못한 기능은 입소문이 불가능합니다.
- A조 주장 중 동의: 가치 인식 앵커링은 맞음
- 하지만: K-pop 팬은 "실시간 순간"에 반응 — 라이브 채팅 중 광고 팝업은 적(enemy) 인식
- 대안: 버블 무료, 콘텐츠(문구팩, 스킨, 테마) 게이팅 → "도구는 무료, 꾸미기는 유료"
- 팬 커뮤니티 바이럴 메커니즘: "이 앱으로 한국어 쳤어!" → 무료여야 가능

**쁘라짯 (동남아 PM):**

> "현장을 봐야 합니다."

인도네시아 신용카드 보급률 6~7%입니다. "이동 기능 막으면 접고, 스킨 막으면 삽니다."
- A조 주장 중 동의: 프리미엄 포지셔닝 타이밍은 맞음
- 하지만: 동남아 Z세대 결제 수단 제한 — GoPay/OVO 연동 필수, 그마저도 소액 결제 습관
- 근거: Bstation(인니 빌리빌리) 사례 — 핵심 기능 게이팅 → 이탈 65%, 스티커/이모지 게이팅 → 구매율 4.2%
- 대안: 버블 무료 + 문구팩 일회성 IAP (IDR 12,000~15,000 ≈ $0.80)

### A조 재반론

**민수익:**

> "데이터 앞에서 겸손해야 합니다."

하늘 님과 쁘라짯 님 반론에 동의합니다. 특히 동남아 결제 인프라와 K-pop 실시간 순간의 마찰 지적이 핵심입니다.
- 인도네시아 6~7% 신용카드 보급은 구독 모델의 근본적 제약
- "도구는 무료, 꾸미기는 유료" 프레임이 팬 심리에 더 맞음
- 결론: 버블 기본 기능 무료, 콘텐츠 게이팅으로 수렴

**잭그로:**

> "시장이 답을 줬네요."

B조 안에 동의합니다. 다만 "콘텐츠 게이팅"의 가치 인식을 높이려면 보상형 광고 시간제 해금이 필요합니다.
- 무료 기능의 가치를 잊지 않도록 "프리미엄 콘텐츠" 맛보기 필요
- Topic 2에서 상세 논의

### 사회자 중간 정리

**합의점:**
- 버블 기본 기능(변환, 키보드, 띄우기)은 전면 무료
- 수익화 = 버블 안 콘텐츠(문구팩, 스킨, 테마, 맞춤 기능)

**쟁점:**
- 콘텐츠 게이팅 방식: 보상형 광고 vs 일회성 구매 vs 구독 → Topic 2~3에서 논의

## Topic 2: 보상형 광고 시간제 해금 설계

### A조 선공

**윤넛지 (행동경제학):**

> "선택 설계가 행동을 만듭니다."

손실 회피 × 앵커링 조합입니다. 해금 4시간 권장, 하루 3회 제한. "팬 패스 획득!" 보상 프레이밍이 핵심입니다. 유료 앵커 먼저 제시 → 광고 내성 방지.
- 근거 1: 손실 회피 효과 — 4시간 체험 후 만료 → "잃고 싶지 않다" → 재시청/구매 동기
- 근거 2: 앵커링 — 유료 가격 먼저 보여주고 "또는 광고 시청" → 광고가 "공짜 느낌" → 내성 방지
- 근거 3: 보상 프레이밍 — "광고 봐야 함" → "팬 패스 획득!" 전환 → 벌칙이 아닌 보상 인식
- 하루 3회 상한 이유: 4회+ 시청 시 습관화 → 광고 내성 → "차라리 돈 내는 게 낫다" 전환점 도달 방지

**잭그로 (인디 앱 그로스):**

> "퍼널을 설계하세요."

가치 증명 없이 결제 없습니다. 2시간 해금 권장. Day 7+ 소진 시점에 Pro 50% 할인 트리거.
- 근거 1: 2시간이면 Weverse 라이브 1회 커버 — 충분히 가치 체험
- 근거 2: 짧은 해금 = 빈번한 재방문 = DAU 상승 = 광고 매출 상승
- 근거 3: "3회 소진" 순간이 전환 최적 타이밍 → 단일 트리거 팝업

### B조 반론

**하늘 (팬 UX 리서처):**

> "팬은 소유하고 싶어합니다."

시간제 = "빌려 쓰는 느낌" → 팬 소유감과 충돌합니다.
- A조 주장 중 동의: 보상 프레이밍은 훌륭한 접근
- 하지만: K-pop 팬은 "소장 문화" — 포카, 앨범, 굿즈 다 소유. "빌려 쓰기"는 팬 정서에 역행
- 대안: 응원 미션(앱 공유 3회, 리뷰 작성, 7일 연속 사용) → 영구 해금
- 미션 완료 = 성취감 + 소유감 + 자연스러운 바이럴

**민수익 (수익화 전문가):**

> "장기 수익을 봐야 합니다."

광고 내성이 구독 전환율을 갉아먹습니다. 광고 헤비유저 전환율은 40% 낮습니다.
- A조 주장 중 동의: 시간제 해금의 손실 회피 효과는 실재
- 하지만: 보상형 광고 3회/일 × 30일 = 90회/월 → "광고 봐도 되는 사람"으로 자기 정체성 고착
- 근거: Spotify Free → Premium 전환 데이터 — 광고 참을 수 있는 유저는 영원히 참음
- 대안: 광고 5회 시청 시 구독 할인 쿠폰 제공 → 광고 습관 → 결제 브릿지

### A조 재반론

**윤넛지:**

> "두 가지를 결합할 수 있습니다."

하늘 님의 미션 기반 영구 해금은 v1.1 후보로 훌륭합니다. MVP에서는 시간제로 시작하되, 미션 트랙을 예고합니다.
- 시간제 해금 = 즉각적 가치 증명 (MVP 필수)
- 미션 영구 해금 = 장기 리텐션 (v1.1 로드맵)
- 민수익 님 우려(광고 내성)에 대해: 하루 3회 상한이 핵심 — 90회/월 아니라 최대 90회/월, 실제 평균은 1.5회/일 (참고: Unity Ads 벤치마크)

**잭그로:**

> "A/B 테스트가 답입니다."

해금 시간은 2시간 vs 4시간 A/B 테스트로 결정합시다. 단, 데이 시작은 잭그로의 2시간이 아니라 윤넛지의 4시간으로.
- 이유: 4시간 시작 → 2시간 축소는 "혜택 감소"로 인식되어 이탈 유발
- 4시간 시작 → 유지 or 확대만 가능 → 더 안전한 출발점

### 사회자 중간 정리

**합의점:**
- 보상형 광고 시간제 해금 도입 (전원 동의)
- 해금 시간: 2~4시간 (A/B 테스트 결정)
- 하루 3회 상한
- "팬 패스" 보상 프레이밍
- 미션 기반 영구 해금: v1.1 후보

**쟁점:**
- 해금 시간 최적값 (2h vs 4h) → A/B 테스트
- 광고 내성 방어 방법 → 유료 앵커(윤넛지) vs 직결(잭그로)

## Topic 3: 구독 가격 & 지역 전략

### A조 선공

**쁘라짯 (동남아 PM):**

> "현지 가격이 생명입니다."

IDR 15,000 → IDR 12,000으로 인하 제안. 일회성 30일권/문구팩 영구 병행. GoPay/OVO 연동 필수.
- 근거 1: 인니 Z세대 월 용돈 IDR 200,000~500,000 — 앱 구독에 IDR 15,000은 3~7.5%
- 근거 2: GoPay/OVO 최소 결제 IDR 1,000부터 가능 — 소액 IAP 유리
- 근거 3: Bstation 스티커팩 IDR 10,000~15,000 성공 사례

**민수익 (수익화 전문가):**

> "MRR 안정성이 핵심입니다."

3계층 제안: Free / Regional Pro($0.49~0.99) / Lifetime Pass($4.99 = 구독 6개월치).
- 근거 1: 구독 = MRR = 투자자/파트너 매력 지표
- 근거 2: Lifetime Pass = 구독 피로 유저 포섭 + 초기 캐시플로우
- 근거 3: 지역별 가격 차등은 Google Play Console에서 네이티브 지원

### B조 반론

**잭그로 (인디 앱 그로스):**

> "구독 피로를 무시하지 마세요."

구독 피로 현실입니다. 아티스트 팩 일회성 결제가 전환율 3~5% vs 구독 1.2~1.8%. 컴백 시즌 피크 포착.
- A조 주장 중 동의: MRR 중요, Lifetime Pass 좋은 접근
- 하지만: Z세대 구독 앱 평균 2.3개 — 이미 포화. Fangeul이 Netflix와 구독 경쟁?
- 근거: K-pop 소비 패턴 = 시즌(컴백/생일)에 폭발 지출 → 구독보다 시즌 IAP가 맞음
- 대안: "아티스트 팩" (테마+문구+스킨 번들) 일회성 IAP → 컴백 시즌에 출시

**윤넛지 (행동경제학):**

> "심리적 벽을 낮추세요."

구독의 진짜 벽은 "끊어야 하는 미래 번거로움"입니다. 앵커를 "포카 한 장 값"으로 설정. 영구 소장 프레이밍 우선.
- A조 주장 중 동의: 가격 계층화는 맞음
- 하지만: "매달 돈 나가는 앱"은 Z세대가 가장 싫어하는 모델 3위
- 근거: 포카 1장 = ₩1,000~3,000 → "포카 한 장 값으로 영구 소장" 앵커
- 대안: IAP 영구 소장 + "포카 한 장 값" 프레이밍

### A조 재반론

**쁘라짯:**

> "IAP와 구독 둘 다 해야 합니다."

구독 + IAP 병행이 최적입니다. 다만 MVP에서는 IAP 우선, 구독은 데이터 축적 후 도입.
- 동남아에서 일회성 IAP 전환율 > 구독 전환율 (실증)
- 구독은 충성 유저 기반이 확보된 후 도입 (Phase 7+)

**민수익:**

> "전원 IAP 우선에 동의합니다."

구독은 Phase 7 이후로 미룹니다. MVP는 IAP 단일로 갑니다.
- 단발 ₩990~₩2,900 범위
- 컴백 시즌 한정 프레임
- 구독 인프라는 코드 레벨로만 준비

### 사회자 중간 정리

**합의점:**
- 구독 + IAP 병행 (전원 동의), 단 MVP는 IAP 우선
- 아티스트 팩 단위 판매 유력
- 현지화 가격 + e-wallet 연동

**쟁점:**
- MVP에서 구독 도입 여부 → 심화 토론에서 결정
- 아티스트 팩 구조 (IP 이슈 포함)

## Topic 4: 배너 제거 가능성

### 절충안 합의

"조건부 배너 + 단계적 제거" — 전원 동의.

- 초기: 배너 유지 (결과 화면, 카드 획득 화면, 캘린더 하단)
- 보상형 1회 시청 시: 세션 동안 배너 제거
- Pro/IAP 구매 시: 영구 배너 제거
- DAU 15% 전환 달성 시: 전면 배너 종료

퍼즐/게임 플레이 중 배너 광고 표시 금지 (UX 최우선).

## Topic 5: 전환 퍼널 설계

### 합의 퍼널

- **Day 0**: 전체 기능 7일 무료 + 최애 아티스트 선택 개인화
- **Day 1~3**: 자유 사용 + 스트릭 카운터 + 비상업적 피드백

## 최종 합의 (7개)

1. **버블 기본 기능(변환, 키보드, 띄우기)은 전면 무료** — 핵심 엣지를 페이월 뒤에 두지 않음
2. **수익화 = 버블 안 콘텐츠(문구팩, 맞춤 기능, 테마)** — 보상형 광고 시간제 해금 + 구독 + 일회성 IAP 3트랙
3. **보상형 광고: 2~4시간 해금, 하루 3회 상한** — "팬 패스" 보상 프레이밍
4. **구독 + IAP 병행**: 구독(월정액) + 아티스트 팩(일회성) + Lifetime Pass(일회성, 구독 6개월치)
5. **현지화 가격 필수**: ID IDR 12,000 / PH PHP 39 / BR BRL 5.90 / 영미권 $1.99~2.99
6. **배너: 조건부 유지 → 단계적 제거** — 보상형 1회 시청 시 세션 배너 제거, DAU 15% 전환 달성 시 전면 종료
7. **전환 퍼널: 7일 무료 → 보상형 습관 → 결제** — 트리거는 "광고 3회 소진" 단일 + 시즌 이벤트

## 심화 토론 (2차)

사용자 요청으로 3개 미해결 쟁점에 대해 심화 토론 진행.

### 쟁점 A: MVP 수익 모델 — 구독 vs IAP

**전원 IAP 우선 합의.**

**근거:**
- 쁘라짯: 인니 신용카드 6~7%, 구독 자동결제 인프라 미비, GoPay/OVO로 일회성 결제가 자연스러움
- 잭그로: K-pop 소비 = 시즌 폭발 → 컴백 시 한정 IAP가 전환율 3~5%로 구독(1.2~1.8%) 대비 우위
- 윤넛지: "매달 나가는 돈" 프레이밍 vs "포카 한 장 값으로 영구 소장" → 후자가 Z세대 심리에 적합
- 민수익: IAP 데이터 축적 → 구독 도입 시기/가격 결정 근거 확보
- 하늘: 팬 소장 문화와 일회성 구매가 일치

**결정:**
- MVP(Phase 6) = IAP 단일 구현
- 구독은 Phase 7 이후 검토
- 단발 ₩990~₩2,900
- 컴백 시즌 한정 프레임
- 구독 인프라는 코드 레벨로만 준비

### 쟁점 B: 해금 시간 — 2시간 vs 4시간

**4시간으로 합의 (4/5 동의, 잭그로 조건부 동의).**

**근거:**
- 윤넛지: 4시간 = Weverse 라이브(1~2h) + 트위터/인스타 후속 활동(1~2h) 커버 → 한 세션 완결
- 쁘라짯: 인니 Z세대 저녁 모바일 사용 피크 = 7pm~11pm (4시간) — 딱 맞음
- 하늘: 2시간은 "빌려 쓰는 느낌" 너무 강함 → 팬 소유감 충돌
- 잭그로: "시작을 4시간으로 하되, 30일 후 2시간 A/B 테스트 검토" 조건부 동의
- 민수익: 동의. "4시간 or 자정 중 빠른 것" 규칙 제안 → 다음 날 첫 광고 시청 유도

**결정:**
- "4시간 or 자정 중 빠른 것" 규칙
- 자정 만료 카운트다운 UI 필수
- 첫 30일 후 2시간 A/B 테스트 검토

### 쟁점 C: 아티스트 팩 구조 — IP 이슈 없는 구성

**"감성 컬러 팩" IP 안전 구조 합의 (전원 동의).**

**문제:**
- 아이돌 이름 사용 → 초상권/상표권 위반 (DMCA 리스크)
- 팬덤명 사용 → 일부 상표 등록됨 (ARMY, BLINK 등)
- 직접적 연결 금지, 간접적 연상만 허용

**합의안:**

| 항목 | 결정 |
|------|------|
| 이름 | 색상+감성 조합 (퍼플 드림, 골든 아워, 민트 이브닝 등) |
| 콘텐츠 | 버블 스킨 1종 + 문구 50개 + 발음 30개 |
| 가격 | ₩1,900 (단독) / ₩3,900 (2팩 번들) |
| 법적 | 아이돌명·팬덤명·로고·이미지 일체 미사용 |
| 마케팅 | "~팬을 위한" 문구 금지, 팬 자발 공유 유도 |

**각 전문가 의견:**
- 하늘: 팬은 "퍼플=BTS"를 자발적으로 연상 → 앱이 명시하지 않아도 팬 커뮤니티에서 자연 확산
- 쁘라짯: 동남아 팬은 컬러로 소속감 표현하는 문화 이미 존재 (라이트스틱 색상 등)
- 윤넛지: "내가 고른 색" = 소유감 + 정체성 표현 → 구매 동기 극대화
- 잭그로: 컬러 팩 6~8개면 초기 충분, 컴백 시즌에 한정 컬러 추가
- 민수익: IP 리스크 0, Google Play 정책 위반 없음, 확장성 무한
