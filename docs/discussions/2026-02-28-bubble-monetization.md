# 플로팅 버블 수익화 전략 패널 토론

> 날짜: 2026-02-28
> 패널: 민수익(수익화), 쁘라짯(동남아PM), 하늘(팬UX), 잭그로(그로스), 윤넛지(행동경제학)

---

## 핵심 합의 (7개)

1. **버블 기본 기능(변환, 키보드, 띄우기)은 전면 무료** — 핵심 엣지를 페이월 뒤에 두지 않음
2. **수익화 = 버블 안 콘텐츠(문구팩, 맞춤 기능, 테마)** — 보상형 광고 시간제 해금 + 구독 + 일회성 IAP 3트랙
3. **보상형 광고: 2~4시간 해금, 하루 3회 상한** — "팬 패스" 보상 프레이밍
4. **구독 + IAP 병행**: 구독(월정액) + 아티스트 팩(일회성) + Lifetime Pass(일회성, 구독 6개월치)
5. **현지화 가격 필수**: ID IDR 12,000 / PH PHP 39 / BR BRL 5.90 / 영미권 $1.99~2.99
6. **배너: 조건부 유지 → 단계적 제거** — 보상형 1회 시청 시 세션 배너 제거, DAU 15% 전환 달성 시 전면 종료
7. **전환 퍼널: 7일 무료 → 보상형 습관 → 결제** — 트리거는 "광고 3회 소진" 단일 + 시즌 이벤트

## Topic 1: 버블 게이팅 타당성

### A조 (민수익, 잭그로) — 버블 게이팅 지지

**민수익:** 버블 Pro 게이팅은 가치 인식 앵커링. 배너 대비 ARPU 3~5배 향상 가능. 킬러 피처를 무료 개방하면 광고 시청 동기 없음.

**잭그로:** 버블은 페인킬러(없으면 고통). 보상형 광고 = 결제 의향 측정 필터. 경쟁 없을 때가 프리미엄 포지셔닝 최적 타이밍.

### B조 (하늘, 쁘라짯) — 버블 게이팅 반대

**하늘:** Weverse 라이브 순간 마찰 → "도움→방해" 인식 역전. 써보지 못한 기능은 입소문 불가. 대안: 버블 무료, 콘텐츠 게이팅.

**쁘라짯:** 인도네시아 신용카드 6~7%. "이동 기능 막으면 접고, 스킨 막으면 산다." 대안: 버블 무료 + 문구팩 일회성 IAP.

### 합의
**A조가 재반론에서 B조 안에 동의** → 버블 기본 무료, 콘텐츠 게이팅으로 수렴.

---

## Topic 2: 보상형 광고 시간제 해금 설계

### A조 (윤넛지, 잭그로) — 시간제 해금 지지

**윤넛지:** 손실 회피 × 앵커링. 해금 4시간 권장, 하루 3회. "팬 패스 획득!" 보상 프레이밍. 유료 앵커 먼저 제시 → 광고 내성 방지.

**잭그로:** 가치 증명 없이 결제 없음. 2시간 해금 권장. Day 7+ 소진 시점에 Pro 50% 할인 트리거.

### B조 (하늘, 민수익)

**하늘:** 시간제 = "빌려 쓰는 느낌" → 팬 소유감 충돌. 대안: 응원 미션(공유, 리뷰, N회 사용) 기반 영구 해금.

**민수익:** 광고 내성이 구독 전환율을 갉아먹음. 광고 헤비유저 전환율 40% 낮음. 대안: 광고 5회 → 구독 할인 쿠폰.

### 합의
- 해금 시간 2~4시간 (A/B 테스트)
- 하루 3회 상한
- "팬 패스" 보상 프레이밍
- 미션 기반 영구 해금은 v1.1 후보

---

## Topic 3: 구독 가격 & 지역 전략

### A조 (쁘라짯, 민수익)

**쁘라짯:** IDR 15,000 → IDR 12,000으로 인하. 일회성 30일권/문구팩 영구 병행. GoPay/OVO 연동 필수.

**민수익:** 3계층: Free / Regional Pro($0.49~0.99) / Lifetime Pass($4.99 = 구독 6개월치). MRR 안정성 핵심.

### B조 (잭그로, 윤넛지)

**잭그로:** 구독 피로. 아티스트 팩 일회성 결제가 전환율 3~5% vs 구독 1.2~1.8%. 컴백 시즌 피크 포착.

**윤넛지:** 구독의 진짜 벽은 "끊어야 하는 미래 번거로움". 앵커를 "포카 한 장 값"으로. 영구 소장 프레이밍 우선.

### 합의
- 구독 + IAP 병행 (전원 동의)
- 아티스트 팩 단위 판매 유력
- 현지화 가격 + e-wallet 연동

## Topic 4: 배너 제거 가능성

### 절충안 합의
"조건부 배너 + 단계적 제거" — 초기 배너 유지, 보상형 1회 시청 시 세션 배너 제거, Pro/IAP 영구 제거, DAU 15% 전환 시 전면 종료.

## Topic 5: 전환 퍼널 설계

### 합의 퍼널
- Day 0: 전체 기능 7일 무료 + 최애 아티스트 선택 개인화
- Day 1~3: 자유 사용 + 스트릭 카운터 + 비상업적 피드백
- Day 4~7: 아티스트 팩 2개+에 보상형 3회/일 적용
- Day 7+: 광고 3회 소진 직후 단일 트리거 결제 팝업
- 이벤트: 컴백·생일 한정 IAP + 스트릭 마일스톤 할인 쿠폰

## 심화 토론 (2차)

### 쟁점 A 합의: MVP = IAP 단일 구현
- 전원 IAP 우선 합의. 구독은 Phase 7 이후
- 단발 ₩990~₩2,900, 컴백 시즌 한정 프레임
- 구독 인프라는 코드 레벨로만 준비

### 쟁점 B 합의: 해금 시간 = 4시간
- "4시간 or 자정 중 빠른 것" 규칙
- 자정 만료 카운트다운 UI 필수
- 첫 30일 후 2시간 A/B 테스트 검토

### 쟁점 C 합의: "감성 컬러 팩" IP 안전 구조
- 이름: 색상+감성 (퍼플 드림, 골든 아워, 민트 이브닝 등)
- 콘텐츠: 버블 스킨 + 문구 50개 + 발음 30개
- 가격: ₩1,900 (단독) / ₩3,900 (2팩)
- 법적: 아이돌명·팬덤명·로고·이미지 일체 미사용
- 마케팅: "~팬을 위한" 문구 금지, 팬 자발 공유 유도
