### "오늘의 팬 미션" 설계
- 데일리 문구 3개 타이핑 챌린지
- 한글 마스킹 → 키패드 타이핑 → 맞추면 공개
- K-pop 캘린더 연동 — 오늘 생일/컴백이면 관련 문구 우선 출제
- 팬 프레이밍 — "학습"이 아닌 "팬 미션/챌린지"
- 완료 시 confetti + 스트릭 카운터
- 수익화: 허니문 무제한 → Day 14 후 1미션 무료 + 추가 보상형
- 배너: 미션 화면 하단 OK (결과/완료 화면)

### 키보드 키 하이라이트
- 기존 커스텀 키패드에 GlobalKey로 각 키 추적 중
- 변환 결과 자모 분해 → 해당 키 순서대로 하이라이트
- HangulEngine.decompose 기존 엔진 활용
- 정답 키 초록 하이라이트, 오답 빨간 흔들림, 완성 시 웨이브 등 가능

## 2. TTS UX 패널 합의

### UI
- **🔊 아이콘 1개** — 문구 카드 하단 액션 바 (즐겨찾기, 복사 옆)
- **탭 = 재생.** 다시 듣고 싶으면 다시 탭.
- **재생 중 피드백**: 아이콘 애니메이션 (음파 펄스)
- 속도 조절 / 구간 반복 / 슬라이더 — **전부 불필요** (v1)

### 캐싱
- 첫 재생 시 다운로드 → 로컬 캐시 → 이후 즉시 재생

### 카운터
- 같은 문구 재재생은 카운트 안 함 (세션 내 재생 ID 추적)
- 새 문구 재생 시에만 ttsPlayCount 증가

### 시간 조작 방어
- 기존 MonetizationState의 단조증가 타임스탬프 + HMAC 서명 인프라 활용
- ttsLastResetDate 같은 패턴

## 3. TTS 보상형 해금 — 기존 결정 변경

### 기존 (3/9 결정)
> "즐겨찾기/TTS 제한 해금 경로 = IAP만 (보상형으로 해금 불가)"

### 변경 (4/8 결정, 만장일치)

| 경로 | TTS 해금 | 즐겨찾기 해금 |
|------|---------|-------------|
| **IAP** (₩990 아무 SKU) | 영구 무제한 | 영구 무제한 |
| **보상형** (1회 시청) | +2회 일시 추가 | 해당 없음 (IAP만) |
| **기본** | 5회/일 | 5슬롯 |

### 세부 규칙
- 보상형 1회 → +2회 (RC `tts_rewarded_bonus` = 2)
- 하루 보상형 3회 제한 유지 → 최대 +6회 → 하루 총 11회
- 5회 소진 시 UX: "오늘 5번 들었어요" + 두 CTA ("광고 보고 2회 더" / "무제한 해금 ₩990")
- IAP는 별도 SKU 없음 — 기존 3-SKU 아무거나 사면 TTS 무제한 포함
- IAP subtitle 업데이트: "+ TTS 무제한" 추가

### 변경 근거 (산술)
- DAU 10K, IAP only: ₩29,700/일
- DAU 10K, 보상형 추가: ₩52,290/일 (1.76배)
- IAP 전환율 30% 하락해도 보상형 수익이 상쇄
- 이탈 방지 741명 → DAU 자체 증가 → 복리 효과
- 동남아 유저: IAP 안 살 유저 98%를 이탈시키기보다 보상형 수익으로 전환
