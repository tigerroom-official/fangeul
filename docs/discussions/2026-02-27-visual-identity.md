# Fangeul 비주얼 정체성 패널 토론 결과

> 2026-02-27 | 패널: 소연(UI/UX), 마크(K-pop마케팅), 하나(브랜딩), 닥터윤(컬러심리), 제이(그로스)

## 최종 결정

### 1. 기본 모드: 다크 + 토글 (Option C)
- 다크 모드 기본값 (K-pop 팬덤 미학 일치)
- 설정에서 라이트/시스템 전환 용이하게
- 순수 블랙(`#000000`) 금지 → 딥 네이비(`#1E1E2E`) 계열
- 텍스트: 순백 대신 `#E8E8F0` (할로 효과 방지, WCAG AA 준수)
- 출시 후 A/B 테스트로 데이터 검증

### 2. 컬러 전략: 팬덤 독립 + 중립 액센트
- 보라색(현재 seedColor) → BTS 연상 리스크 → **변경 필요**
- 어떤 아이돌 그룹 공식 컬러도 메인으로 사용 금지
- 딥 네이비/소프트 차콜 베이스 + Fangeul 고유 액센트
- 팬덤 컬러는 콘텐츠 레이어(공유 카드 테마)에서만 선택적 제공

### 3. 디자인 톤: "문화적 프리미엄" — 절제된 일상, 폭발하는 모먼트
- 도구 화면(변환기, 키패드, 설정): 차분하고 인지 부하 낮은 프리미엄
- 보상 화면(스트릭, 카드, 공유): Lottie 애니메이션 + 에너제틱
- 대비가 감동을 만든다 — 차분한 배경 위에서 축하 폭죽이 진짜 축제

### 4. 타이포그래피
- MVP: Pretendard Subset(2~3MB, 한글 2,350자+라틴) → 키패드·자모·공유 카드
- 일반 텍스트: `GoogleFonts.notoSansKr()` 동적 로드
- APK 증가 5MB 이내 유지

### 5. 공유 카드: "절제된 임팩트"
- 9:16 비율, 큰 한글 중앙 배치
- 다크/라이트 2종 + 팬덤 컬러 액센트 6개 선택지
- Lottie 파티클: v1.1에서 추가
- 핵심 원칙: **한글이 주인공, 나머지는 조연**

## 핵심 원칙 요약

| 원칙 | 설명 |
