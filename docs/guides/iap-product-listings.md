## 등록 절차

1. Play Console → 앱 선택 → **수익 창출** > **인앱 상품**
2. **"제품 만들기"** 클릭
3. 상품 ID 입력 (예: `fangeul_theme_custom_color`) — **생성 후 변경 불가**
4. 태그 입력
5. 기본 언어(ko) 이름/설명 입력
6. **"번역 관리"** → 각 언어별 이름/설명 입력
7. 가격 설정 → KRW 기본 → "가격 변환 적용" → 동남아 수동 조정
8. **"활성"** 상태로 전환
9. 3개 상품 반복
